# Players — fichas por categoría

Registro interno de todos los actores relevantes del ecosistema. Se actualiza cada semana tras el pass de descubrimiento (ver `discovery-playbook.md`). **Nunca** se crea página pública automáticamente: solo cuando un player tiene 3+ menciones sustanciales + ángulo editorial propio + hueco real en búsqueda.

## Formato de entrada

```
### {slug}
- Nombre: {nombre oficial}
- Categoría: {categoría}
- Geografía primaria: {market}
- Geografías activas: {list}
- Tipo de flota servida: {fleet-types clásicos}
- Primera aparición: YYYY-MM-DD en Nº N
- Última aparición: YYYY-MM-DD en Nº N
- Menciones sustanciales: N
- Fuentes de noticias: {lista corta}
- Estado de página pública: no-candidato | candidato | pendiente | pública
- Página pública: /players/{slug}/ (si existe)
- Nota editorial: {una línea de ángulo propio}
```

## Cómo se mantiene

- El pass semanal de descubrimiento (ver `discovery-playbook.md`) añade players nuevos con ficha mínima.
- Los "candidato Q3/Q4" solo pasan a creación de página tras acumular 3+ menciones sustanciales y tener un ángulo editorial redactable (300+ palabras sin relleno).
- Ningún player pasa a página pública sin aprobación humana explícita.
