# Desde Pulpo — novedades para la próxima edición

Este archivo alimenta la sección **"Desde Pulpo"** de cada edición semanal.

Si al momento de generar la edición **no hay nada útil aquí**, la sección se omite.
No rellenar con texto comercial genérico.

---

## Cómo escribir un update

Formato recomendado para cada entrada:

```
### Título corto (máx 10 palabras)
- Fecha: YYYY-MM-DD
- Tipo: feature | caso | testimonial | artículo | evento | dato | release
- País(es): MX | ES | USA | LATAM | Global
- Resumen (2–4 líneas claras, sin adjetivos huecos)
- Por qué importa al lector de Radar Fleet (1 línea)
- Enlace: https://...
```

Cuando haya varias entradas, la edición elegirá **una sola** (la más fuerte) para no saturar.
El resto queda como material para futuras ediciones.
